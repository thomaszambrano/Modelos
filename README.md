## Ejercicios de prueba con modelos de regresion y clasificacion lineal.


para ejecutar los ejercicios, se debe tener instalado python y las siguientes librerias:
- numpy
- pandas
- scikit-learn
- matplotlib

para instalarlas, se puede usar el siguiente comando:

```bash
pip install numpy pandas scikit-learn matplotlib
```

Hacemos pruebas con datos de prueba y validacion para verificar el correcto funcionamiento de los modelos.

La idea es entender como funcionan los modelos de regresion y clasificacion lineal, y como se pueden usar para predecir y clasificar datos.
